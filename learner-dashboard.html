Django implementation notes:
Dashboard view at /dashboard/ — login required. Data populated from: LearnerProfile, Enrolment, LessonProgress, Session, Achievement models.
Progress ring: calculated server-side as (lessons_complete / lessons_total × 100). Skill area bars: per-category lesson completion aggregated in view context.
Upcoming sessions: queryset filtered by session.date ≥ today, ordered by date, limited to 3. "Add to calendar" → iCal file download.
Recommended next steps: algorithm based on last-accessed course + learning goals from LearnerProfile.
Learning journey stages: defined as milestone thresholds in settings — configurable by admin.
Streak: calculated from daily LearnerActivity log. Achievements: awarded automatically by Django signals on completion events.
Notifications badge: unread count from Notification model. Messages badge: unread Message count.
